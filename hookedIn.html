HookedIn

Platform for UT Coding Bootcamp alumni to share their experiences in the workforce and connect with past cohorts.

  • Node.JS
  • JQuery
  • JavaScript
  • SASS
  • Flexbox
  • Bootstrap
  • HTML5
  • MySQL

The purpose of HookedIn is to have a readily accessible data resource center for UT Bootcamp students and grads. With HookedIn, students and alums can network across cohorts, review portfolios, connect through LinkedIn and email, or find a mentor. The application is fully mobile responsive. The cohesive and professional UI is familiar to other social networking sites.

Landing page features include client and server side form validation, passport validation, and password encryption. Users are given access to the site using a Passport.js token.

First time users are routed to the profile page to add additional optional information. Profile fields are autopopulated for the user's convenience and validated before making changes to the database.

The dashboard is the focal point of the application. User's can search through members by name, cohort, city, and state without refreshing the page. A user can click on any other user in the dashboard. A modal displays with additional information on the selected user. On desktop and tablet views, pie charts are displayed to the page for user employment rates and for the length of time job seeking.

Future changes include an admin interface to select who may or may not sign up to view the app as well as delete accounts upon request. Users will also be able to update and reset their password. Given the current limitations of the database host, weekly database screenshots are only created when the database is running.
