A La Code is a Wordpress site designed to chronicle the lives and tech interests of two web developers in Austin. The site not only shows the design and development skillset of Jillian and Ana, but also fills a current void in the developer-blog space. Many popular tech blogs were stark and lacked personality. A La Code in turn features lifestyle and developer content in a punny icecream branded package

From a development perspective, A La Code is and was a great opportunity to dive into WordPress custom theme development and PHP. A La Code uses WordPress for its popularity as a CMS, plugin potential, and extensive documentation. As a CMS, WordPress and the chose theme, make it simple for administrators at various experience levels to add and edit site content. This functionality is not immediately useful for A La Code, but gives the site potential to expand.

The site uses a custom theme based off of the popular Kale and Underscore themes. With a reliable theme foundation, the site could focus on custom styling, images, and content.

A La Code is a fun and quickly growing site. With WordPress and PHP, there is still a lot to achieve. Next steps include adding functionality to the post and page templates that give non-developer admins more editing controll. Current pages primarily rely on PHP variables and HTML static content. Post content may be generated with the WordPress WYSIWYG post panel.
