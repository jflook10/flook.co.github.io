Mini JS

Small and ongoing frontend JavaScript challenges and projects.

  • ES6
  • JavaScript
  • Flexbox
  • CSS
  • HTML5

Mini JS is a personal project to chronicle JavaScript concepts. Each tile is a small refresher, challenge, and/or exercise focusing on vanilla JS. Projects are small and intentionally limited to one HTML page. I keep my notes and some prior tests in the JS file for later reference.

In addition to the core purpose of JavaScript practice, Mini JS is a way for me to test new visual expressions and styles. Using the HTML5 UP theme, I can focus on the JS exercise and incorporating my personal interest in art history.
